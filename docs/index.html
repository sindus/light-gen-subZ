Desktop app · offline

Subtitles from any video or audio, generated locally.

light-gen-subZ transcribes speech with a local whisper.cpp model and writes a standard .srt file with accurate timestamps — no upload, no account, no cloud.

  • OfflineYour media never leaves your machine
  • MultilingualAutomatic language detection
  • Standard output.srt, works in any editor
  • NativeSmall, fast Tauri + Rust app
  • Optional cloud STTGroq, OpenAI, Deepgram, AssemblyAI
  • Optional translationLocal NLLB-200, DeepL, OpenAI, Google, Azure
curl -fsSL https://raw.githubusercontent.com/sindus/light-gen-subZ/main/install.sh | sh

Detects your architecture and installs a .deb (Debian/Ubuntu) or .AppImage automatically. Requires ffmpeg to be installed and glibc ≥ 2.38 (Ubuntu 24.04+, Debian 13+, Fedora 39+).

brew tap sindus/light-gen-subz && brew install --cask light-gen-subz

Apple Silicon only for now. Requires Homebrew and ffmpeg (brew install ffmpeg).

Uninstall

Linux (.deb): sudo apt remove light-gen-sub-z
Linux (AppImage): rm ~/.local/bin/light-gen-subz.AppImage
macOS: brew uninstall --cask --zap light-gen-subz

See the README for how to also remove downloaded models and settings.
